# raspberry-zwave
Raspberry PI Z-Wave board php Interface


## How to Use

```php
require_once('zwave-api.php');

$zwa = new ZWaveAPI();
$zwa->login_username = '';
$zwa->login_password = '';
```
This is a user details setup on the main Interface,
[http://localhost:8083/smarthome/#/admin](http://localhost:8083/smarthome/#/admin)
